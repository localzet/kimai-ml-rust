# Политика безопасности — kimai-ml

Если вы обнаружили уязвимость или проблему безопасности в этом проекте, пожалуйста, сообщите нам конфиденциально — не создавайте публичные issue до согласованного раскрытия.

Куда сообщать:

- Email: security@localzet.com
- GitHub Security Advisories: предпочтительно через интерфейс GitHub

Пожалуйста, включите в сообщение:

- Короткое описание уязвимости и её потенциальные последствия.
- Конкретные версии/коммиты/ветки, которые затронуты.
- Шаги воспроизведения или PoC, если есть.
- Информацию об окружении: ОС, версии node/npm/другого ПО.
- Любые логи, конфигурации и предложенные обходные пути.

Наши обязательства:

- Подтверждение получения отчёта: в течение 3 рабочих дней.
- Ответ с планом действий или временным решением: обычно в течение 14 дней.
- Публикация исправления и уведомление после согласованного раскрытия.

Не публикуйте подробности уязвимости публично до выпуска исправления.

Репозиторий: https://github.com/localzet/kimai-ml
